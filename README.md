## 🧠 Core Concept
IETE AI Bot is designed to respond based on the user's cognitive approach:

* **Engineering Brain 🧮** Focused on technical precision, mathematical derivations, circuit analysis, and structured problem-solving.
* **General Brain 💡** Provides simple, clear, and intuitive explanations for everyday queries.

---

## 🛠️ Tech Stack & Run Locally
This app is built with **Node.js** and deployed via **Vercel**.

### Prerequisites:
* Node.js installed on your machine.

### Setup Instructions:
1.  **Install dependencies:**
    `npm install`
2.  **Set the API Key:**
    Create a `.env.local` file and add your key: `GEMINI_API_KEY=your_key_here`
3.  **Run the app:**
    `npm run dev`
